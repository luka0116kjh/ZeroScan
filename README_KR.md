# 웹 보안 스캐너 (백신)

[![English](https://img.shields.io/badge/lang-English-blue?style=flat-square)](README.md)
[![Korean](https://img.shields.io/badge/lang-한국어-red?style=flat-square)](README_KR.md)

Flask 기반의 웹 보안 감사 도구로, 대상 URL의 보안 헤더, HTTPS 강제 적용 유무 및 일반적인 성능 지표를 분석합니다. 자동화된 AI 스타일 분석 결과를 제공하며, 스캔 결과에 따른 맞춤형 보안 학습 모듈을 추천합니다.

## 주요 기능

- **보안 헤더 분석**: 필수 보안 헤더(HSTS, CSP, X-Frame-Options 등)의 설정 여부를 점검합니다.
- **HTTPS 검증**: 대상 사이트가 암호화된 연결을 강제하는지 확인합니다.
- **성능 벤치마킹**: 응답 시간을 측정하여 보안 모니터링 가시성을 확보합니다.
- **규칙 기반 AI 분석**: 긍정적인 보안 요소와 주요 보안 위험에 대한 직관적인 요약을 제공합니다.
- **맞춤형 학습 경로**: 스캔 발견 사항을 토대로 특정 보안 모듈을 추천합니다.
- **스캔 로그 및 통계**: 과거 기록을 추적하고 등급 분포 등 대시보드 형태의 요약을 제공합니다.

## 사용 기술

- **백엔드**: Python, Flask, Requests
- **분석 로직**: 규칙 기반 점수 산정(0-100점) 및 등급제(A-F)
- **데이터 보관**: JSONL 기반 로컬 로깅 시스템

## 시작하기

### 1. 패키지 설치

```powershell
pip install flask requests
```

### 2. 서버 실행

```powershell
python .\백신\app.py
```

기본적으로 `http://127.0.0.1:5000`에서 서버가 구동됩니다.

## API 엔드포인트

- `GET /`: 메인 UI를 제공합니다.
- `GET /api/learn`: 보안 학습 모듈 카탈로그를 반환합니다.
- `GET /api/logs/summary`: 과거 스캔 결과에 대한 통계 요약을 제공합니다.
- `POST /scan`: 특정 URL을 감사합니다.
  - 바디 예시: `{"url": "https://example.com"}`

## 프로젝트 구조

```text
백신/
|-- app.py             # 메인 앱 로직 및 API 경로
|-- data/              # 스캔 로그 보관 (scan_logs.jsonl)
|-- templates/         # UI용 HTML 템플릿
`-- static/            # CSS/JS 정적 파일 (해당되는 경우)
```

## 보안 고려 사항

- **SSRF(서버 측 요청 위조) 방지**: 사설/내부 IP 대역 및 로컬호스트 스캔을 차단하는 로직이 포함되어 있습니다.
- **리다이렉트 제한**: 오픈 리다이렉트 루프를 방지합니다 (최대 5회).
- **타임아웃 관리**: 리소스 고갈을 막기 위해 모든 요청에 기본 타임아웃을 적용합니다.

## 참고

이 도구는 교육 및 기초 감사 목적을 위해 설계되었습니다. 흔히 발생하는 설정 오류를 짚어줌으로써 웹 보안 인식을 높이는 "백신" 역할을 합니다. 전문적인 모의해킹이나 종합적인 취약점 스캐너를 완전히 대체하지는 않습니다.
